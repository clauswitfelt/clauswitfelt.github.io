# Cascading Style Sheets - typografiark.

CSS er et program, til at layoute websider, meget mere detaljeret end i HTML, hvor man jo kun kan skabe simple layoutmæssige ting.
CSS er et omfattende emne og at blive god til alt i CSS tager lang tind. 
Vi vil her kun komme ind på nogle få, men centrale ting.

# Ord og begreber
HTML-ordrer kalder vi tags.
Når vi bruger CSS siger vi, at vi "styler"
CSS tilføjer ekstra muligheder til vores basale HTML.

## 3 måder at inkludere CSS
1. Man kan style een tag fx 

```
<body style="background-color:powderblue;">
```

<body></body> er en standard-HTML-tag, men CSS'en: style="background-color:powderblue;", kvalificerer denne kommando og tilføjer er powderblue-farve. Det er en ret besværlig måde at style på!

2. Man kan skrive al sin CSS i en STYLE-sektion, fx i toppen af sit HTML-dokument

```
<STYLE>
h1
{
  color: green;
}

</STYLE>

```

Dette læses sådan: I style-sektionen står der en række tags/HTML-kommandoer, som bliver redefineret/stylet.
I eksemplet ovenfor er det kun Headline 1, H1 og det eneste ekstra, der gøres ved det er det, der står mellem de krøllede parenteser - at tekstfarven sættes til grøn.

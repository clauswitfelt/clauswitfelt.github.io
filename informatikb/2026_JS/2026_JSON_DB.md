


## Hvad er et relationel database?
En relationel database er en database, der består af flere forskellige tabeller med forbindelser (relationer) imellem. 
Dette sikrer god integritet (=datakvalitet) og mindsker redundans (gentagne data).

Undersøg bilbasen (http://www.bilbasen.dk ). Er der tale om en relationel database og i så fald hvorfor?

## Hvad er JSON eller CSV?
JSON står for _JavaScript Object Notation_ og betyder blot en struktur, man kan gemme data, fx data fra en webrequest eller en database.

I kan se eksempler på JSON-data her: https://www.w3schools.com/js/js_json.asp 

https://thecodingtrain.com/tracks/data-and-apis-in-javascript/data/1-client-side/4-json

CSV betyder blot komma-separeret fil og er endnu et dataformat, hvor man blot adskiller værdierne med kommaer
